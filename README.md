# Автотесты API Яндекс.Прилавок.

Проект содержит автотесты для проверки API сервиса аренды самокатов Яндекс.Самокат.
Тесты написаны на Python с использованием Pytest и библиотеки requests.

## Что тестируется
Проверяются основные сценарии работы API заказов:

1. Создание нового заказа
2. Получение трек-номера заказа
3. Проверка успешного ответа сервера (код 201/200)
4. Получение данных созданного заказа
5. Корректность статуса заказа

## Состав проекта
- configuration.py - адрес сервера и пути API
- data.py - тестовые данные (тела запросов)
- sender_stand_request.py - функции отправки HTTP-запроcов
- tests/test_orders.py - автотесты заказов
- README.md
- .gitignore - список игнорируемых файлов


## Требования
- Python 3.9+
- Установленные пакеты pytest и requests

## Установка зависимостей
Выполнить команду в терминале: python -m pip install -r requirements.txt

## Запуск тестов
1. Открыть терминал в корневой папке проекта
2. Выполнить команду: python -m pytest -v
3. Дождаться завершения выполнения тестов и просмотра результатов

## Результаты тестирования
Если тесты завершаются со статусом PASSED, это означает:
- сервер корректно принимает запросы;
- заказы успешно создаются;
- API работает стабильно;
При статусе FAILED — обнаружен дефект API или ошибка запроса.
